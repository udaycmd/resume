## Uday Tiwari - Resume

---

This repository contains the LaTeX source code and build automation for my professional resume.

### Build Instructions

You need a LaTeX distribution installed. [TinyTeX](https://yihui.org/tinytex/) is recommended for a lightweight footprint.

1.  **Install LaTeX dependencies:**

    ```bash
    make install
    ```

2.  **Generate the PDF:**

    ```bash
    make
    ```
